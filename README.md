# 📖 Quranic – A Modern & Premium Islamic Companion

![Quranic Banner](assets/banner_readme.png)

**Quranic** is a feature-rich, high-performance open-source application designed for Muslims worldwide. It offers a beautiful and seamless experience for reading the Quran, tracking prayer times, and much more—all built with a modern tech stack focused on speed and simplicity.

---

## ✨ Features

- **📖 Digital Mushaf**: Highly readable, swipable Quran pages using modern rendering.
- **🕋 Prayer Times (Solah)**: Accurate calculation for your location including Adhan notifications.
- **🕌 Hijri Calendar**: Seamlessly transitions between Gregorian and Hijri dates.
- **🧭 Qiblah Finder**: Precise direction finding using your device's compass.
- **🤲 Daily Adhkaar**: Curated morning, evening, and various other supplications.
- **🎧 Audio Recitation**: Stream and listen to beautiful Quran recitations.
- **🔍 Advanced Search**: Quickly find any Ayah or Surah.

---

## 🛠️ Tech Stack

Quranic is split into two main components: a feature-packed mobile application and a sleek web frontend.

### 📱 [Mobile App](mobile/)

- **Framework**: [Expo](https://expo.dev/) (React Native)
- **Language**: TypeScript
- **Styling**: Native components and custom design system
- **Icons**: Lucide React Native
- **Libraries**: Adhan (Prayer Times), Moment Hijri, Expo Audio, Reanimated.

### 🌐 [Frontend Web](frontend/)

- **Framework**: [Next.js 15+](https://nextjs.org/)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 🚀 Getting Started

To get the project running locally, follow these steps:

### Prerequisites

- Node.js (v18 or higher)
- npm or pnpm

### Clone the Repository

```bash
git clone https://github.com/musamusakannike/quranic.git
cd Quranic
```

### Run the Mobile App

```bash
cd mobile
npm install
npx expo start
```

### Run the Frontend

```bash
cd frontend
npm install
npm run dev
```
