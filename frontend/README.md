# 🌐 Quranic Web Frontend

![Frontend Banner](../assets/banner_readme.png)

Quranic's web frontend is built with **Next.js 15+** and **Tailwind CSS 4**. It provides a fast, lightweight experience for reading the Quran and accessing essential Islamic information online.

---

## 🛠️ Tech Highlights

- **[Next.js 15+](https://nextjs.org/)**: The best of React for production.
- **[Tailwind CSS 4](https://tailwindcss.com/blog/tailwindcss-v4-alpha)**: Modern, utility-first CSS Framework (Alpha release).
- **[Framer Motion](https://www.framer.com/motion/)**: Sleek web animations and interactions.
- **[Lucide React](https://lucide.dev/)**: High-quality vector icons for the web.

---

## 🚀 Get Started

1. **Install Dependencies**
   ```bash
   npm install
   # or
   pnpm install
   ```
2. **Start the Development Server**
   ```bash
   npm run dev
   # or
   pnpm dev
   ```
3. **Open [http://localhost:3000](http://localhost:3000)** with your browser to see the result.

---

## 📁 Directory Structure

- `app/`: Next.js App Router pages and layouts.
- `public/`: Static assets and icons.
- `components/`: Modular React components.
- `styles/`: Global CSS and Tailwind configuration.
